ReConstruct

ReConstruct is a desktop F# WPF application for 3D volume reconstruction, which is an area of Computational Geometry.

Essentially it takes a Signed Distance Field and as input and outputs an extrapolated 3D surface.

Though I initially started this project to learn Radiological imaging techniques, the focus has shifted towards specializing in generic 3D reconstruction which has applications in many field of Science and Engineering such as terrain mapping, 3D printing, robotics and manufacturing, games, etc. Certainly a worthy undertaking.

On UI side, I needed a very lightweight scaffolding that allowed me to focus on functionality and manipulate UI elements with total freedom, without the overhead or limitations of MVVM. For this reason, I used an MVC-ish approach that is easy to reason about, allows quick UI iteration and is also light on resources.

Rendering is done with OpenGL and utilizes different strategies to achieve good performance like async/parallel processing, buffer pooling, minimizing heap allocations, SIMD, etc.
